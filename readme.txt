Easy integration of the TikTok Pixel for WooCommerce.

== Description ==
**TikTok Pixel - E-Commerce**
★★★★★

TikTok is one of the most popular apps. Are you the operator of a website with WooCommerce and want to track your conversations? Or other events? No problem! Please note that you need a TikTok Business account - otherwise this plugin won't do you much good!

### Best Features ###

* **Easy Setup**
Install add u're TikTok Pixel and Done!

* **Event Tracking (PageView - Browse)**

* **WooCommerce Event Tracking (ViewContent)**

* **WooCommerce Event Tracking (AddToCart)**

* **WooCommerce Event Tracking ( PlaceAnOrder )**

* **WooCommerce Event Tracking (Purchase)**
With parameters like product name, currency, quantity, price and total price! Important for skaling your ads on TikTok!

### Planned / Roadmap ###

* **Custom Event Tracking**
Track your own Events like registration, viewForm, clickForm, SubmitForm etc.

* **Search Event Tracking**
We would like to monitor the search field as well.
